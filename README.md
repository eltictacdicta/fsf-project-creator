# FSF Project Creator

Script para crear proyectos automáticamente con DDEV y FS-Framework.

## Características

- Configura automáticamente DDEV con PHP 8.3
- Clona el repositorio FS-Framework
- Organiza la estructura del proyecto
- Configura el entorno de desarrollo listo para usar

## Requisitos

- Git
- DDEV
- Bash

## Uso

```bash
./crear-poyecto-fsf.sh
```

El script te pedirá el nombre del proyecto y hará todo automáticamente.

## Después de crear el proyecto

```bash
cd nombre-del-proyecto
ddev start
```
